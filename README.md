
# PapePrinter - Smart Printing Layout Tool
PapePrinter is a web application designed to simplify the process of printing images and calculating costs.  
It provides an intelligent layout engine that adapts image orientation, supports special modes (ID cards, stickers), and calculates ink density to estimate real printing costs.  

### Features
- Responsive interface built with HTML5, CSS3, and JavaScript.
- Smart orientation adjustment for images to maximize paper usage.
- Special modes:
  - **ID Mode**: Generate grids for official ID sizes.
  - **Sticker Mode**: Create mosaics based on real centimeter sizes.
- Cost calculation engine:
  - Ignores white pixels to measure ink density.
  - Calculates paper + ink costs automatically.
  - Monochrome discount option (75% off).
- Export options: Save layouts as PDF or image using html2canvas and jsPDF.

This tool is designed for students, professionals, and print shops who want a practical way to optimize printing layouts and pricing.
